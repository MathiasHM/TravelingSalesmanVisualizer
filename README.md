# TSP Algorithm Comparator (WIP)

A WinForms (Windows only, sorry) hobby project for visualising and comparing Traveling Salesman Problem algorithms side by side in real time.
Just use "Randomize" to generate a node set, or click the canvas to place and remove nodes manually, then run it. Each algorithm runs on its own dedicated thread when solving for accurate timing comparisons. Progress updates stream to the UI in real time while the algorithms are still running.


## Build requirements
- .NET 10
- WinForms (Again, I am truly sorry, I had to learn it for school)
